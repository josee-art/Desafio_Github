# Desafio\_GITHUB - LASI# \# Calculadora de IMC em Python# # Sistema de cálculo de IMC desenvolvido em Python com funcionalidades de:# # \- cadastro de pacientes;# \- cálculo automático de IMC;# \- classificação corporal;# \- atualização de peso;# \- histórico de evolução;# \- salvamento em arquivos `.txt`.# # \---# # \# Funcionalidades# # \## Cadastro de pacientes# # Permite cadastrar:# # \- nome;# \- peso;# \- altura.# # Exemplo:# # ```text# Carlos | 80 Kg | 1.74 m# ````# # \---# # \## Cálculo automático de IMC# # O sistema calcula automaticamente o IMC usando a fórmula:# # ```text# IMC = peso / altura²# ```# # Exemplo:# # ```text# IMC: 26.42# ```# # \---# # \## Classificação do IMC# # O programa classifica o paciente em:# # | IMC            | Classificação  |# | -------------- | -------------- |# | Menor que 18.5 | Abaixo do peso |# | 18.5 até 24.9  | Peso normal    |# | 25 até 29.9    | Sobrepeso      |# | 30 ou mais     | Obesidade      |# # \---# # \## Mostrar pacientes cadastrados# # Lista todos os pacientes cadastrados mostrando:# # \* nome;# \* peso;# \* altura;# \* IMC;# \* classificação.# # \---# # \## Atualização de peso# # Permite atualizar o peso do paciente.# # Ao atualizar:# # \* o IMC é recalculado;# \* o histórico é salvo automaticamente;# \* os dados são atualizados.# # \---# # \## Histórico de evolução# # O sistema salva o histórico completo do paciente.# # Exemplo:# # ```text# Carlos | 80 Kg | IMC: 26.42 | Sobrepeso# Carlos | 70 Kg | IMC: 23.12 | Peso normal# ```# # \---# # \## Histórico individual# # Permite visualizar o histórico de apenas um paciente específico.# # \---# # \## Salvamento em arquivos# # O sistema utiliza dois arquivos:# # \### `pacientes.txt`# # Armazena os pacientes atuais.# # \### `historico\_imc.txt`# # Armazena toda evolução dos pacientes.# # \---# \# 📂 Estrutura do Projeto# # ```text# 📁 Projeto\_IMC# │# ├── Calculadora\_IMC.py# └── README.md# ```#
